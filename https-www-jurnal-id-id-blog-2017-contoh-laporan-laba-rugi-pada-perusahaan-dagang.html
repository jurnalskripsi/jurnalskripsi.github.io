14. Jelaskan apa yang dimaksud dengan Network id dan host id dan berikan contohnya


Pada dasarnya Network ID sendiri dapat kita artikan sebagai suatu Identitas dari suatu kelompok IP Address atau subnet, yang mana Network ID ialah suatu bagian dari IP address yang menunjukan letak network atau jaringan dari suatu IP Address. Sedangkan untuk Host ID sendiri ialah suatu nomor perangkat keras fisik Ethernet Network Interface Card atau yang sering disingkat dengan sebutan NIC.

Sebagai salah satu contohnya ialah:

Kelas A yaitu 10.11.12.1 maka untuk contoh ini network ID nya ialah 10. sedangkan untuk Host ID ialah 11.12.1.Kelas B yaitu 188.10.11.1 maka untuk contoh ini network ID nya ialah 188.10. sedangkan untuk Host ID ialah 11.1.Kelas C yaitu 192.168.1.1 maka untuk contoh ini network ID nya ialah 192.168.1 sedangkan untuk Host ID ialah 1.

Pembahasan

Pada umumnya Net ID sendiri dapat berperan penting dalam mengidentifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network, yang mana pada konteks ini dapat kita simpul kan bahwa seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama.

Dalam materi pelajaran Ilmu Pengetahuan Sosial, terutama dalam jenjang SMA pasti akan diajarkan sebuah materi pelajaran bernama Sosiologi. Sosiologi merupakan ilmu yang membahas mengenai apa yang terjadi pada saat ini, khususnya pada pola hubungan dalam masyarakat, serta berusaha mencari pengertian secara umum, rasional, empiris, dan bersifat umum.

Menurut pengertian lainnya, sosiologi merupakan sebuah ilmu yang mempelajari tentang perilaku sosial masyarakat, baik antara individu dengan individu, individu dengan kelompok, ataupun kelompok dengan kelompok.

Karena manusia adalah makhluk sosial, maka tidak akan pernah jauh dengan yang namanya hubungan sosial. Karena bagaimanapun juga hubungan-hubungan tersebut berpengaruh besar terhadap perilaku sebagai orang.

Sebagai ilmu pengetahuan sosial yang objeknya masyarakat, sosiologi memiliki empat macam fungsi, fungsi sosiologi dalam penelitian, pembangunan, perencanaan sosial dan pemecahan masalah sosial. Berikut masing-masing pengertian dan penjelasannya.

30. perbedaan net id dan host id


Net ID berperan dalam identifikasi suatu network dari network yang lain, sedangkan host ID berperan untuk identifikasi host dalam suatu network. Jadi, seluruh host yang tersambung dalam jaringan yang sama memiliki net ID yang sama. Garis pemisah antara bagian network dan host tidak tetap, bergantung kepada kelas network. IP address dibagi ke dalam lima kelas, yaitu kelas A, kelas B, kelas C, kelas D dan kelas E.

31. Pengertian network id, host id, dan broadcast id


Pengertian✍️

Network id

Network ID adalah bagian dari IP address yang menunjukkan di jaringan mana komputer tersebut berada.

Host id

Host adalah identitas dari host (interface router, workstation, dan device) yang terhubung ke jaringan.

Broadcast id

broadcast adalah alamat jaringan yang digunakan untuk mengirim ke semua perangkat yang terhubung ke jaringan komunikasi akses ganda.
