15. Pengaruh mapel matematika pada mapel penjaskes


Matematika dapat memberikan pengaruh positif pada pelajaran olahraga dan kesehatan (penjaskes) karena dapat membantu siswa untuk memahami konsep-konsep yang digunakan dalam olahraga dan kesehatan. Beberapa contohnya adalah:

Siswa dapat menggunakan konsep pengukuran dan pemetaan untuk menganalisis dan mengevaluasi performa atletik mereka sendiri atau orang lain.
Siswa dapat menggunakan konsep statistik untuk menganalisis data hasil tes kondisi fisik dan menentukan tingkat kesehatan diri sendiri atau orang lain.
Siswa dapat menggunakan konsep pemodelan matematika untuk menjelaskan dan memprediksi perilaku sistem tubuh manusia dalam berolahraga atau menjaga kesehatan.
Siswa dapat menggunakan konsep geometri dan trigonometri untuk menganalisis dan menerapkan gerakan tubuh dalam berolahraga, seperti dalam mengevaluasi teknik dalam berbagai macam olahraga
Jadi dengan mapel matematika yang dikuasai oleh siswa dapat membantu siswa memahami konsep di dalam mapel penjaskes, yang akan membuat mereka lebih mudah memahami dan menguasai materi tersebut

16. contoh soal sifat asosiatif dan distributif kelas 5 sd mapel matematika


Kelas : V (5 SD)
Materi : bilangan bulat
Kata Kunci : bilangan, bulat, sifat-sifat, asosiatif, distributif

Pembahasan :
1. Penjumlahan
Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

(a + b) + c = a + (b + c)

contoh :

(2 + 6) + 8 = 2 + (6 + 8)

⇔ 8 + 8 = 2 + 14

⇔ 16 = 16


2. Pengurangan

tidak bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c tidak berlaku:

(a – b) – c ≠ a – (b – c).

contoh :

(3 - 5) - 7 ≠ 3 - (5 - 7)

⇔ -2 - 7 ≠ 3 - (-2)

⇔ -9 ≠ 5


3. Perkalian

Bersifat asosiatif, sehingga untuk sembarang bilangan bulat a, b, dan c, berlaku :

(a x b) x c = a x (b x c)

contoh :

(2 x 5) x 6 = 2 x (5 x 6)

⇔ 10 x 6 = 2 x 30

⇔ 60 = 60.


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

a x (b + c) = (a x b) + (a x c)

contoh : 

–5 x [9 + (–1)] = [–5 x 9] + [–5 x (–1)] 

⇔ –5 x 8 = –45 + 5

⇔ –40 = –40


ii. distributif terhadap pengurangan

a x (b – c) = (a x b) – (a x c)

contoh :

7 x [–10 – 12] = [7 x (–10)] – [7 x 12]

⇔ 7 x -22 = -70 - 84

⇔ -154 = -154.


4. Pembagian

Tidak bersifat asosiatif, sehingga Untuk sembarang bilangan bulat a, b, dan c berlaku:

(a : b) : c ≠ a : (b : c)

contoh :

(8 : 4) : 2 ≠ 8 : (4 : 2)

⇔ 2 : 2 ≠ 8 : 2

⇔ 1 ≠ 4


Bersifat distributif, sehingga untuk sembarang bilangan bulat a, b, dan c berlaku :

i. distributif terhadap penjumlahan

(a + b) : c = (a : c) + (b : c)

contoh : 

(12 + 6) : 2 = (12 : 2) + (6 : 2)

⇔ 18 : 2 = 6 + 3

⇔ 9 = 9


ii. distributif terhadap pengurangan

(a – b) : c = (a : c) – (b : c)

contoh :

(21 – 9) : 3 = (21 : 3) – (9 : 3)

⇔ 12 : 3 = 7 - 3

⇔ 4 = 4

 

Semangat!

18. Tugas mandiri .Matematika SD.Pendidikan S1 PGSDSoal:1. Buat kajian tentang sistem bilangan Real...​


Jawaban:

Sistem bilangan real adalah salah satu konsep fundamental dalam matematika. Ini mencakup semua bilangan yang dapat diwakili dalam bentuk titik-titik pada garis bilangan yang tak terbatas. Berikut ini adalah kajian singkat tentang sistem bilangan real:

1. **Definisi**: Bilangan real adalah himpunan semua bilangan rasional dan irasional. Ini termasuk semua bilangan bulat, pecahan, desimal, serta akar kuadrat dan kubik yang tidak dapat diungkapkan sebagai pecahan sederhana.

2. **Representasi**: Bilangan real dapat direpresentasikan dalam bentuk titik-titik pada garis bilangan real. Semua titik pada garis ini mewakili bilangan real tertentu, dan garis tersebut tidak memiliki batasan, sehingga mencakup semua bilangan yang ada.

3. **Bilangan Rasional vs. Bilangan Irasional**: Bilangan real dapat dibagi menjadi dua kategori utama, yaitu bilangan rasional dan bilangan irasional. Bilangan rasional adalah bilangan yang dapat dinyatakan sebagai pecahan dua bilangan bulat (contohnya, 1/2, 3/4), sedangkan bilangan irasional tidak dapat diekspresikan dalam bentuk pecahan sederhana (contohnya, √2, π).

4. **Operasi Matematika**: Bilangan real mengikuti semua aturan operasi matematika yang umum, seperti penjumlahan, pengurangan, perkalian, dan pembagian. Operasi ini dapat dilakukan antara bilangan real, menghasilkan bilangan real lainnya.

5. **Sifat-sifat Khusus**: Bilangan real memiliki berbagai sifat unik, seperti totalitas (setiap dua bilangan real dapat dibandingkan), tak terbatas (tidak ada bilangan real terbesar atau terkecil pada garis bilangan real), serta sifat-sifat lainnya yang digunakan dalam analisis matematika.

6. **Penggunaan dalam Ilmu dan Keperluan Sehari-hari**: Bilangan real memiliki aplikasi yang luas dalam berbagai cabang matematika, sains, teknik, ekonomi, dan bidang lainnya. Mereka juga digunakan dalam kehidupan sehari-hari untuk mengukur, menghitung, dan memodelkan fenomena alam dan kehidupan manusia.

Sistem bilangan real merupakan dasar penting dalam matematika dan memiliki peran kunci dalam pemahaman dan pemecahan masalah dalam berbagai bidang. Keberadaan bilangan real yang tak terbatas memberikan keragaman dan fleksibilitas yang diperlukan dalam pemodelan berbagai fenomena dalam dunia nyata.

Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
