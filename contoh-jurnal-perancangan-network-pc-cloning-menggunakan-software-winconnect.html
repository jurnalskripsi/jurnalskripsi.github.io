17. jaringan terbagi atas 2 network yang terkoneksi ke internet dimana setiap network terdiri atas 10 pc


Jaringan terbagi atas 2 network yang terkoneksi ke internet dimana setiap network terdiri atas 10 pc merupakan salah satu jaringan yang menggunakan topologi tree.

Pembahasan

Topologi jaringan merupakan suatu hal yang menjabarkan hubungan geometris antara unsur-unsur dasar penyusun sebuah jaringan seperti node, link, dan station. Pemilihan topologi jaringan pada umumnya didasarkan pada skala jaringan, biaya, tujuan, dan penggunaan.

Topologi tree merupakan  jenis topologi jaringan yang dilakukan dengan  menggabungan beberapa topologi star yang terhubung dalam topologi bus. Kelebihan Topologi Tree diantaranya  topologi ini mudah dikembangkan menjadi jaringan yang lebih luas, susunan datanya terpusat berdasarkan hirarki. Sedangkan kekurangan Topologi Tree adalah jika perangkat komputer yang menduduki tingkatan tertinggi pada topologi ini mengalami masalah, maka tingkatan di bawahnya akan ikut bermasalah serta topologi ini memiliki kinerja jaringan terbilang lambat.

Tools untuk Source Code

Karena memang pekerjaan utamanya adalah menuliskan kode, tentunya tools pada bagian ini tidak boleh dikompromikan. Apa saja tools yang dapat membantu kita dalam meningkatkan produktifitas penulisan kode?

Editor / IDE

Pertama pastinya editor atau IDE yang digunakan untuk menuliskan kode. Untuk IDE sendiri sifatnya sangat spesifik bahasa, karena setiap bahasa pastinya memiliki fitur-fitur khusus yang perlu juga dikembangkan implementasi khususnya. Jadi saya hanya akan membahas beberapa editor yang direkomendasikan di sini.

Rekomendasi utama saya untuk editor: vim dan Sublime Text. Beberapa orang lain merekomendasikan emacs juga.

Pencarian File

Selama karir pemrograman anda, pernakah anda bingung dalam mencari sebuah file lama yang entah disimpan di mana? Lupa nama file pastinya? grep, sebuah tools yang sudah sangat tua, namun belum tergantikan dapat membantu anda. Jika belum tau kegunaannya, pelajari sekarang juga!

Jika berada di Windows, coba Power Grep.

Version Control

Programmer mana yang tidak menggunakan version control? Sebelumnya saya telah menulis sedikit tentang version control, dan biasanya saya tidak perlu lagi menjelaskan kenapa perangkat lunak jenis ini sangat penting bagi programmer.

Diff and Merge

Semua programmer pasti pernah melakukan penggabungan file yang 99% sama, dengan hanya beberapa baris yang berbeda. Atau mungkin anda perlu mencari perbedaan nilai dari sebuah file teks berukuran 100MB. Untuk masalah-masalah seperti ini, anda dapat menggunakan diff tools, yang biasanya juga dipaketkan dengan version control.

KDiff3 adalah salah satu tool yang sering dipaketkan dengan version kontrol, jadi tentunya sangat direkomendasikan. Diff and Merge milik Perforce juga sangat bagus, jika butuh fitur tambahan seperti perbandingan gambar.

Lint

Apa itu lint)? Mengutip Wikipedia,

In computer programming, lint was the name originally given to a particular program that flagged some suspicious and non-portable constructs (likely to be bugs) in C language source code. The term is now applied generically to tools that flag suspicious usage in software written in any computer language.

Jadi kira-kira software lint itu berperan untuk membaca kode program kita, dan kemudian mencari pola-pola umum kode yang dapat berbahaya, misalnya berpotensi menghasilkan lobang keamanan atau bug.

Tapi karena semua bahasa pemrograman berbeda-beda, tentunya tools yang dimiliki untuk lint juga berbeda-beda. Misalnya, untuk Javascript ada JSLint. Yang penting adalah, setelah mengetahui keberadaan software ini, kedepannya kita dapat mencari software sejenis untuk bahasa yang sedang digunakan, dan mudah-mudahan jumlah bug dapat berkurang karena diingatkan oleh lint :)

Refactoring

Jika sudah bekerja pada suatu kode atau program sangat lama, cepat atau lambat kita akan menemukan bahwa kode yang dituliskan selama ini telah menjadi besar, kacau, dan tidak terawat. Istilah kerusakan kode karena terlalu banyak perubahan dan perbaikan sejak masa awal pengembangan dikenal dengan nama “Technical Debt”.

Technical Debt merupakan sesuatu yang tidak dapat dihindari, karena banyak permasalahan dalam pengembangan perangkat lunak yang hanya ditemukan ketika menuliskan kode. Pengalaman dan perencanaan sebanyak apapun tidak dapat menggantikan penulisan kode langsung. Hal ini jugalah yang menyebabkan saya tidak mempercayai orang-orang yang berkata kemampuan menulis kode tidak begitu penting dalam pengembangan perangkat lunak.

Anyway (jadi ngelantur :p).

Tools refactoring berguna untuk membantu kita dalam mengurangi technical debt dengan memberikan fitur-fitur untuk memperbaiki kode yang sudah tua, seperti membuat fungsi / method dari baris kode yang berulang kali dituliskan atau mengganti nama kelas menjadi lebih deskriptif.

Seperti lint, tentu saja tools ini terikat dengan bahasa pemrogramannya. Contohnya, untuk C# terdapat Resharper yang legendaris. Tools untuk masing-masing bahasa biasanya selalu bisa didapatkan dengan mudah dan gratis. Jadi, tunggu apa lagi? Segera cari tools refactoring untuk bahasa favorit anda!

Profiler

Jika menemukan masalah performa dalam program anda, apakah yang pertama kali anda lakukan? Mencoba menebak di bagian mana program berjalan paling lambat? Mencoba mengubah beberapa bagian sesuai dengan intuisi?

32. aplikasi software yang menghubungkan antara komputer dengan Network selain WiFi dan LAN adalah


Sistem Operasi
Software pertama yang harus dan wajib kita install pada komputer kita sebelum mengakses internet adalah sistem operasi komputer. Ya, sistem operasi komputer adalah salah satu software mutlak yang wajib dimiliki oleh semua user, agar komputer itu sendiri bisa berjalan dan juga bekerja dengan optimal.

Sistem operasi, sesuai dengan namanya merupakan sebuah software yang dikembangkan dan juga diinstall pada sebuah komputer, yang memiliki kegunaan untuk membantu funsi operasional pada sebuah komputer. Dengan begitu, sebuah komputer bisa dengan mudah digunakan oleh usernya.

Fitur yang terdapat pada sebuah sistem operasi menyangkut kompatibilitas terhadap berbagai macam software untuk mengakses internet lainya, yang akan dibahasa pada point berikutnya. Saat ini, paling tidak ada 3 buah atau tiga jenis sistem operasi komputer yang populer dan juga banyak digunakan, yaitu sistem operasi Microsoft Windows, Linux, dan juga Mac OS. Macintosh. (baca juga: jenis sistem operasi komputer)

Web Browser
Setelah sistem operasi sudah terinstall dan dipastikan dapat beroperasi sesuai dengan ketentuan yang ada, maka saatnya kita menginstall web browser. Web browser adalah sebuah aplikasi atau software yang sangat penting bagi anda yang akan melakukan akses ke dalam internet.

Web browser menjadi salah satu software yang paling berperan atas penggunaan internet. Anda menggunakan web browser ini untuk melakukan browsing dan surfing ke berbagai macam situs yang tersedia di dalam jaringan internet. Dengan menggunakan web browser, maka anda akan benar-benar dapat merasakan manfaat penting dari koneksi internet di dalam komputer anda.


Web browser pun tersedia berbagai macam jenis. Beberapa macam web browser yang populer antara lain adalah Google Chrome, Mozilla Firefox, Internet Explorer, dan Safari Browser. Dari jenis web browser yang ada tentnunya browser terbaik menjadi incaran para user untuk mengoptimalkan keperluan user melakuakn browsing.

Network Driver
Hal lainnya yang tidak kalah penting untuk mengakses internet adalah network driver. Network driver merupakan sebuah software yang bertugas untuk menjalankan hardware yang berhubungan dengan kemampuan komputer dalam terhubung ke sebuah jaringan. Karena salah satu caara menghubungkan komputer dengan internet adalah dengan melengapi perangkat untuk mengakses internet baik dari hardware maupun software.

Jadi, tanpa adanya network driver ini, maka setiap perangkat keras jaringan di dalam komputer tidak akan berfungsi dengan normal. Biasanya, network driver sudah menjadi satu paket bersama dengan laptop ataupun komputer yang dirakit, dan disesuaikan dengan jenis dan juga tipe network hardware yang digunakan pada komputer tersebut.

Email Client
Email merupakan salah satu fitur yang sangat bermanfaat di dalam sebuah jaringan komputer, terutama jaringan internet. Dengan menggunakan email, maka kita bisa berkirim surat dengan sangat mudah dan cepat dengan teman-teman dan juga relasi di seluruh dunia.


Email client sendiri merupakan sebuah software stand alone, yang berfungsi untuk menerima, mengirim, dan mengedit suatu pesan elektronik. Biasanya kita bisa menginstall email client ini di dalam komputer kita. salah satu email client yang pernah bersinar di masanya adalah Microsoft Outlook Express. Prefix = /10 --> Maka ada 10 digit biner 1 pada submask

Submask = 11111111.11000000.00000000.00000000

                = 255.192.0.0

y = banyak digit 0 pada oktet terakhir

Jml host per subnet (termasuk network dan broadcast)

= 2^y

= 2^22

= 4.194.304
