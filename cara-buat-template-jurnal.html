30. bagaimana cara menampilkan dokumen surat dari template?


Cara pertama, melalui jendela Microsoft Word 2007 yang telah terbuka. Klik New > New from existing. Browse dan buka file BIODATA.dotx yang telah Anda buat. Hasilnya dokumen baru dengan desain yang telah Anda buat akan ditampilkan dan siap digunakan.

￼

Cara kedua, melalui Windows Explorer (My Documents atau My Computer). Buka Windows Explorer Anda, browse dan klik dua kali atau klik kanan > New pada file BIODATA.dotx, maka jendela program Microsoft Word akan tampil dengan sebuah dokumen baru yang siap dipakai.

￼


Anda cukup mengisi data seperlunya dan menyimpannya dengan nama file baru. Dokumen siap dicetak dan diarsip sesuai kebutuhan.

Contoh template di atas sangat sederhana, Anda dapat menggunakan teknik di atas untuk membuat template dokumen yang lebih kompleks, misalnya surat perjanjian, news letter produk baru, atau bahkan booklet dan majalah.

38. cara membuat template pada microsoft word?​


Jawaban Dengan Langkah²

Dokumen perusahaan tidak melulu berisikan angka-angka dan tulisan. Ada kalanya gambar atau graphis pun perlu ada untuk menggambarkan besaran penjualan misalnya. Nah untuk memudahkan biasanya bagian administrasi sudah memiliki template sendiri bila ingin menyertakan gambar ke dalam laporan perusahaan.

Template Microsoft Word adalah dokumen Word yang menyertakan pengaturan khusus seperti teks boilerplate, macros, header dan footer, kamus khusus, toolbar, dan entri AutoText. Elemen-elemen yang sama akan selalu ada setiap kali Anda membuka template dan tidak dapat diubah. Dengan adanya template Anda juga sudah memotong durasi pengaturan file menjadi lebih cepat karena sudah ada template standar yang hanya perlu update tanggal maupun angka-angka nominal di dalamnya.

Merencanakan Template Word

Sebelum Anda membuat template Word Anda, buat daftar fitur yang ingin Anda sertakan di dalamnya seperti contoh di bawah ini:

Teks yang akan selalu disertakan dalam dokumen

Memformat kolom, margin, tab berhenti, catatan akhir, dan catatan kaki

Makro untuk mengotomatisasi tugas.

Bidang tanggal yang diperbarui secara otomatis setiap kali templat dibuka.

Alamat dan informasi kontak

Fields atau AutoText untuk informasi yang berubah seperti nomor halaman, judul dokumen, atau path file di header dan footer.

Teks placeholder dengan pemformatan tertentu seperti ukuran jenis huruf.

Setelah Anda menyusun semua elemen yang akan disertakan ke dalam template, buatlah prototipe dalam dokumen Word kosong dan lakukan penyesuaian bila dibutuhkan, dan jangan lupa simpan prototipenya.

Membuat template di Word 2019, 2016, 2013, DAN 2010

1. Buka File

2. Pilih Save As

3. Di kotak dialog Save As, di kotak teks File Name, ketik nama template

4. Kotak dialog Microsoft Word Save As menyimpan templat kustom

5. Jalur file berubah ke lokasi templat default. Templat di folder ini muncul di kotak dialog saat Anda membuat dokumen baru dari templat. Namun, Anda dapat memilih folder lain jika Anda mau

6. Pilih Save.

Membuat template di Word 2007

Anda bisa membuat template dengan cara berikut:

1. Pilih tombol Microsoft Office

2. Pilih Save As > Word Template

3. Di kotak dialog Save As, untuk menyimpan templat di folder templat default, di bawah Favorite Links pilih Template. Ini adalah folder yang disarankan karena templat di folder ini muncul di kotak dialog Template ketika Anda membuat dokumen baru berdasarkan templat. Namun, Anda juga dapat menavigasinya ke folder lain jika Anda mau.

4. Ketikkan nama deskriptif untuk template, lalu Save.

Membuat template di Word 2003

1. Pilih File > Save As

2. Ketikkan nama yang dapat dikenali untuk templat lalu Save as type dan pilih Document Template

3. Pilih Save.

Dokumen Anda sekarang disimpan sebagai templat dengan ekstensi file .dot atau .dotx dan dapat digunakan untuk membuat dokumen baru. Praktis bukan? Kini Anda tinggal mengganti konten secara reguler sesuai template yang sudah ada. Sama praktisnya ketika saat Anda membutuhkan dokumen tersebut dicetak kemudian Anda menggunakan jasa PrinterQoe yang merupakan solusi cetak dokumen online di seluruh wilayah Indonesia.
