4. apa yang dimaksud software pengolah kata dan software pengolah gambar dan berikan contohnya


Software Pengolah kata adalah software yang digunakan untuk memproduksi/ mengolah kata (mengedit, memformat, menyusun, hinga mencetak)

Software pengolah gambar adalah software yang difungsikan untuk memproduksi/ mengolah gambar, baik gambar statis maupun gambar dinamis.
contohnya: Corel Draw, Adobe Photoshop.
