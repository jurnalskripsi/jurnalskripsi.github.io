7. Contoh Soal Dari Metode Gabungan Eliminasi Dan Substitusi


Carilah nilai x dan y dari persamaan berikut dengan cara eliminasi
4x + 3y = 345x + y   = 37

Jawab :


Pertama, kita akan mencari nilai variabel x. Untuk mengeliminasi variabel x, maka persamaan nomer 1 (atas) dikalikan dengan 1 dan persamaan nomor dua (bawah) kita kalikan dengan 3. Kedua persamaan dikurangkan agar variabel y hilang.


4x + 3y = 34 | X1 → 4x + 3y = 34 5x + y = 37 | X3 → 15x + 3y = 111 ______________ - -11x = -77 x = 7

Setelah kita mendapat nilai variabel x, kita akan mencari variabel y dengan cara yang tak jauh beda.


4x + 3y = 34 | X5 → 20x + 15y = 170 5x + y = 37 | X4 → 20x + 4y = 148 ______________ - 11y = 22 y = 2


Jadi kita dapat bahwa nilai x = 7 dan y = 2

Langkah I (eliminasi variabel y)Untuk mengeliminasi variabel y, ingat koefisien y harus sama, sehingga persaman 3x + 2y = 12 dikalikan 1 dan persamaan 2x – y = 8 dikalikan 2, maka:3x + 2y = 12        │× 1 =>3x + 2y = 122x – y = 8            
3x + 2y = 124x – 2y = 16  │× 2 =>4x – 2y = 16
---------------  
+7x + 0 = 28x = 28/7x = 4
Langkah II (eliminasi variabel x)Untuk mengeliminasi variabel x, ingat koefisien x harus sama, sehingga persaman 3x + 2y = 12 dikalikan 2 dan persamaan 2x – y = 8 dikalikan 3, maka:3x + 2y = 12        │× 2 =>6x + 4y = 242x – y = 8            
6x + 4y = 246x – 3y = 24  │× 3 =>6x – 3y = 24

33. contoh soal metode subtitusi dan eliminasi beserta penjelasan​


Jawaban:

Tentukan nilai variabel x dan y dari persamaan berikut menggunakan metode substitusi

2x + 4y = 28 ... (i)

3x + 2y = 22 ... (ii)

Penyelesaian:

1# Memilih salah satu persamaan yang akan dipindahkan salah satu variabel-nya

Hal pertama yang dilakukan saat menggunakan metode substitusi yaitu memilih salah satu persamaan untuk dipindahkan elemen-nya. Disarankan memilih persamaan yang paling mudah, sehingga tidak menghasilkan angka desimal saat langkah berikutnya. Untuk beberapa kasus setiap persamaan mungkin mempunyai tingkat kesulitan yang sama, yaitu sama-sama menghasilkan angka desimal. Jadi, pemilihan persamaan bersifat bebas dan relatif.

Misalnya dipilih persamaan (i) yaitu 2x + 4y = 28

2# Memindahkan salah satu variabel pada persamaan yang dipilih

Misalnya, dipilih variabel y untuk dipindahkan ke ruas kanan,

2x + 4y = 28 ... (i)

⇔ 2x = 28 - 4y

Karena, dipilih variabel y untuk dipindahkan, sehingga diperoleh bentuk solusi untuk variabel x, yaitu menghilangkan koefisien x dengan membagi masing-masing ruas dengan nilai koefisien x,

2x = 28 - 4y

2 2

⇔ x = 14 - 2y ... (iii)

Sehingga ditemukan persamaan (iii) bentuk solusi dari variabel x

3# Menggabungkan persamaan (iii) pada persamaan yang tidak dipilih di awal (ii) untuk menghitung solusi numerik variabel lainnya

3x + 2y = 22 ... (ii)

Karena diperoleh bentuk solusi x pada persamaan (iii),

x = 14 - 2y ... (iii)

Selanjutnya gabungkan dengan cara mengganti variabel x sebagai bentuk solusinya pada persamaan (ii),

3 x + 2y = 22

⇔ 3 (14 - 2y) + 2y = 22

⇔ 42 - 6y + 2y = 22

⇔ 42 - 4y = 22

⇔ -4y = 22 - 42

⇔ -4y = -20

⇔ -4y = -20

-4 -4

⇔ y = 5

Sehingga, diperoleh solusi variabel y = 5

4# Menghitung solusi numerik variabel lain

Karena sudah ditemukan solusi variabel y = 5, dapat dihitung dengan menggabungkan y = 5 pada bentuk solusi x pada persamaan (iii)

x = 14 - 2y ... (iii)

⇔ x = 14 - 2(5)

⇔ x = 14 - 10

⇔ x = 4

36. contoh reaksi dari substitusi, eliminasi ,adisi?


Berikut contoh-contoh reaksi dari substitusi, eliminasi, dan adisi.

Substitusi:

CH₄ + Cl₂ → CH₃Cl + HCl (disebut juga klorinasi)

CH₃CH₂Cl + AgOH → CH₃CH₂OH + AgCl

NaOH + HCl → NaCl + H₂O

CH₃COOCH₃ + H₂O → CH₃COOH + CH₃OH

Eliminasi:

CH₃CH₃ → CH₂CH₂ + H₂ (katalisator Pt)

CH₃CH₂Cl → CH₂CH₂ + H₂O + KCl (katalisator KOH)

CH₃CH₂OH → CH₂CH₂ + H₂O (suhu 180°C, katalisator H₂SO₄)

Adisi:

CH₂ = CH₂ + H₂ → CH₃ - CH₃

          ||                                  |

CH₃ - C - CH₃ + H₂ → CH₃ - CH - CH₃

CH₂ = CH - CH₃ + Br₂ → CH₂ - CH - CH₃

                                        |        |

                                       Br     Br

CH₂ = CH - CH₃ + H-Cl → CH₃ - CH - CH₃

                                                    |

                                                   Cl

Pembahasan

Reaksi organik utama dapat dikelompokkan atas 6 macam, yaitu substitusi, adisi, eliminasi, kondensasi, redoks, dan polimerisasi.

Mari kita bahas tentang substitusi, eliminasi, dan adisi sesuai persoalan yang ditanyakan.

A. Substitusi

Substitusi artinya penukaran atau penggantian. Reaksi substitusi adalah reaksi di mana terjadi penukaran atau penggantian suatu atom atau gugus atom oleh atom (gugus atom) yang lain.

Contoh-contoh reaksi substitusi:

Halogenasi alkana, atau pembuatan alkil halida dari alkana, di mana atom H diganti oleh atom halogen.

CH₄ + Cl₂ → CH₃Cl + HCl (disebut juga klorinasi)

Pembuatan alkohol dari alkil halida, di mana atom halogen diganti oleh gugus -OH.

CH₃CH₂Cl + AgOH → CH₃CH₂OH + AgCl

Netralisasi NaOH + HCl → NaCl + H₂OHidrolisis ester CH₃COOCH₃ + H₂O → CH₃COOH + CH₃OH

B. Eliminasi

Eliminasi artinya penghilangan. Reaksi eliminasi adalah reaksi di mana terjadi penghilangan beberapa atom atau gugus atom untuk mendapatkan zat baru. Pada reaksi eliminasi terjadi perubahan ikatan jenuh menjadi takjenuh.

Dehidrogenasi alkana CH₃CH₃ → CH₂CH₂ + H₂ (katalisator Pt) Dehidrohalogenasi alkil halida CH₃CH₂Cl → CH₂CH₂ + H₂O + KCl (katalisator KOH)Dehidrasi alkohol CH₃CH₂OH → CH₂CH₂ + H₂O (suhu 180°C, katalisator H₂SO₄)

C. Adisi

Adisi artinya penambahan. Reaksi adisi adalah reaksi di mana terjadi pengubahan ikatan takjenuh menjadi jenuh dengan cara menangkap atom-atom lain. Jadi, reaksi adisi hanya dapat dialami oleh zat-zat yang mengandung ikatan takjenuh (ikatan rangkap).

Hidrogenasi CH₂ = CH₂ + H₂ → CH₃ - CH₃

Adisi terhadap senyawa karbonil, yaitu

         O                                OH

          ||                                  |

CH₃ - C - CH₃ + H₂ → CH₃ - CH - CH₃

Halogenasi

CH₂ = CH - CH₃ + Br₂ → CH₂ - CH - CH₃

                                        |         |

                                       Br      Br

Hidrohalogenasi

CH₂ = CH - CH₃ + H-Cl → CH₃ - CH - CH₃

                                                    |

                                                   Cl

Perhatikan, pada reaksi yang terakhir berlaku Hukum Markovnikov, yaitu atom H dari asam halida cenderung masuk pada atom C ikatan rangkap yang mengikat H lebih banyak. Jadi dari reaksi terakhir di atas, atom H dari HCl menuju ke CH₂, sedangkan atom Cl menuju ke CH.
