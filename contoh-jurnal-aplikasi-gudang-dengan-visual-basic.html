FUNGSI DAN KEGUNAAN VISUAL BASIC

FUNGSI DAN KEGUNAAN VISUAL BASIC

Microsoft Visual Basic for Applications (VBA) adalah sebuah turunan bahasa pemrogramanVisual Basic yang dikembangkan oleh Microsoft dan dirilis pada tahun 1993, atau kombinasi yang terintegrasi antara lingkungan pemrograman(Visual Basic Editor)dengan bahasa pemrograman(Visual Basic)yang memudahkan user untuk mendesain dan membangun program Visual Basic dalam aplikasi utama Microsoft Office, yang ditujukan untuk aplikasi-aplikasi tertentu. VBA didesain untuk melakukan beberapa tugas, seperti halnya mengkustomisasi sebuah aplikasi laiknya Microsoft Office atau Microsoft Visual Studio. Kegunaan VBA adalah mengotomatisasi pekerjaan. Pekerjaan yang dimaksud adalah pekerjaan yang dilakukan secara berulang-ulang dan pekerjaan yang kompleks. VBA berbeda dengan Microsoft Visual Basic, Microsoft Visual Basic memberi banyak pemrograman dan fungsi tingkat lanjut hingga Microsoft Visual Basic dapat dihasilkan program yang lebih kompleks untuk sistem operasi Microsoft Windows maupun Office. Sedangkan VBA hanya dapat dibangun pada aplikasi utama Microsoft Office mengendalikan fungsi aplikasi tersebut melakukan serangkaian objek terprogram. Versi VBA terbaru saat ini adalah versi 6.3 yang dirilis pada tahun 2001, yang mendukung semua program dalam Microsoft Office, yakniMicrosoft Excel, Microsoft Access, Microsoft Word, Microsoft Outlook, Microsoft FrontPage, sertaMicrosoft PowerPoint dan juga Microsoft Visual Studio.

26. wadah untuk aplikasi Visual Basic disebut​


Jawaban:

Wadah untuk aplikasi Visual Basic disebut Form. Dalam Visual Basic, Form adalah objek yang digunakan untuk membuat tampilan antarmuka pengguna (user interface) dari sebuah aplikasi. Form berfungsi sebagai wadah atau kontainer untuk menampilkan elemen-elemen antarmuka pengguna seperti tombol, label, kotak teks, gambar, dan sebagainya.

Dengan menggunakan Form dan kontrol-kontrol Visual Basic lainnya, Anda dapat membuat antarmuka pengguna yang interaktif dan menarik untuk aplikasi Anda. Anda dapat menggabungkan logika program dengan elemen-elemen antarmuka pengguna untuk mencapai tujuan fungsionalitas yang diinginkan.

Contoh:

Buka aplikasi Visual Studio dan buat proyek baru dengan menggunakan template Windows Forms Application.Di jendela desain (Design view) Form default, tambahkan beberapa kontrol seperti tombol (Button), kotak teks (TextBox), dan label (Label) dengan mengklik dan menyeret dari Toolbox ke Form.Ubah properti masing-masing kontrol sesuai kebutuhan, seperti menentukan teks pada tombol atau label, mengatur nama kontrol, dan sebagainya.Double-click pada tombol untuk membuka Code view dan menambahkan kode program yang akan dijalankan ketika tombol diklik. Misalnya, kita dapat menambahkan kode berikut untuk menampilkan pesan pada kotak teks saat tombol diklik:

----------------------------------------------------

Private Sub Button1_Click(sender As Object, e As EventArgs) Handles Button1.Click

   TextBox1.Text = "Tombol diklik!"

End Sub

----------------------------------------------------

JAGUARESPORT Terpelajar

Microsoft Visual Basic adalah sebuah aplikasi yang digunakan untuk melakukan pengembangan berbagai macam aplikasi. Selain itu, environment yang ditawarkan dalam pembangunannya adalah dengan menggunakan Integrated Development Environment atau lebih sering disebut dengan IDE. Selain itu, aplikasi ini memiliki bentuk visual yang akan membantu kita dalam proses pembuatan program.

Visual Basic sendiri adalah sebuah aplikasi yang pada awalnya menggunakan Bahasa pemrogramman BASIC dimana Bahasa BASIC tersebut digunakan untuk melakukan pengembangan aplikasi perangkat lunak berbasis user interface dengan sangat cepat. Kemudian, aplikasi ini menyediakan komponen-komponen yang akan mendukung berbagai jenis project yang ada di Visual Basic. Berikut adalah jenis project-project yang ada pada Visual Basic

Jenis project pada Visual Basic

- Standard EXE

Merupakan jenis aplikasi yang digunakan untuk membangun kepentingan bisnis yang biasanya ada dan tidak harus menggunakan aplikasi-aplikasi lainnya. Dengan menggunakan Standard EXE apa yang digunakan juga sangat sederhana dan terdapat keunggulan dimana semua komponen yang terdapat dapat pada Standard EXE diakui oleh setiap unit komputer yang tersedia meskipun user tersebut bukanlah seorang administrator.

- Active EXE

Active EXE merupakan salah satu project dengan fungsi yang difokuskan untuk aplikasi lain untuk menjadi sebuah layanan service pada aplikasi tersebut. Kemudian, service yang dibuat ini dapat digunakan didalam komputer yang sama maupun komputer yang dioperasikan secara online dengan menggunakan remote automation. Untuk service ini, biasanya tidak terdapat desain user interface dan project dari ActiveX memiliki komponen-komponen yang dapat melakukan komunikasi dengan semua aplikasi yang terdapat dan serta dapat digunakan oleh sistem operasi windows.

- ActiveX DLL

Active DLL adalah jenis aplikasi yang digunakan untuk aplikasi lainnya yang kemudian menjadi bagian dari service. Berbeda dengan Active EXE, hanya terdapat pada satu komputer yang digunakan pada ActiveX DLL saja. Serta aplikasi ini juga tidak menyediakan desain dari user interface. Tetapi, aplikasi ini dapat dijalankan melalui aplikasi Microsoft Transaction Server apabila ingin dijalan pada komputer lainnya.

- ActiveX Control

Merupakan salah satu komponen yang digunakan pada device yang memakai dan mendukung aplikasi dari ActiveX dan berupa sebuah aplikasi.

- VB Application Wizard

Project dari aplikasi ini akan membantu pengguna aplikasi untuk melakukan pembuatan aplikasi dengan sangat mudah.

- Addin

Project yang seperti Standard EXE yang memiliki lebih banyak kebebasan

- Data project

Data project didesain untuk pembuatan aplikasi berbentuk database

- DHTML

DHTML digunakan untuk melakukan pembuatan aplikasi internet dengan sistem client side sehingga dapat digunakan pada client dengan metode serta fungsi-fungsi yang terdapat pada DHTML

- IIS application

Seperti DHTML, tetapi sistem yang digunakan adalah server side. Untuk komponen yang dimiliki IIS juga menggunakan komponen dari CGI atau lebih dikenal dengan Common Gateway Interface.

- VB Wizard Manager

Pada VB Wizard Manager, aplikasi yang dibuat biasanya digunakan untuk pembangunan sebuah kumpulan akan informasi yang akan disajikan kepada pengguna untuk pembangunan sebuah aplikasi yang kemudian disebut dengan wizard.

- ActiveX Document DLL

Untuk ActiveX Document DLL project yang diberikan adalah pembangunan untuk sebuah aplikasi berbentuk Document Active. Biasanya, pada aplikasi ini menggunakan ekstensi berupada DLL

- Active Document Exe

Tidak beda jauh seperti DLL, hanya saja format ekstensinya adalah Exe.

- VB Enterprise Edition Control

VB Enterprise Edition Control adalah jenis yang tidak jauh berbeda dengan Standard EXE, hanya saja perbedaannya adalah semua fungsi serta tool ya

Ayo Kita Bahas Satu Persatu Artinya Ya.

1. Imports System

Imports System Berarti Kita Mengimpor Semua Kelas, Fungsi Dan Konstanta / Variabel Didalam Namespace system Yang Sudah Disediakan Oleh NET. Framework.

2. Module ModuleKu

Module ModuleKu Berarti Kita Mendefinisikan Module Yang Bernama "ModuleKu". Module Ini Berisi Semua Variabel Dan Fungsi Yang Kita Butuhkan.

3. Sub Main()

Ini Berarti Kita Mendefinisikan Fungsi Main() Didalam Module ModuleKu. Sub Main() Artinya Fungsi Utama. Program Akan Berjalan Mulai Dari Sini.

4. Console.WriteLine("Hello World!")

Console.WriteLine("Hello World!") Berarti Kita Akan Menulis Output "Hello World!" Ke Konsol.

5. End Sub

End Sub Menandakan Akhir Dari Fungsi Yang Kita Buat, Dalam Hal Ini Adalah Fungsi Main().

6. End Module

End Module Menandakan Akhir Dari Module Yang Kita Buat, Dalam Hal Ini Adalah Module ModuleKu.
