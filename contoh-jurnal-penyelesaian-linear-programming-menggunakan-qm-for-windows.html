1. Masalah Transportasi dapat diselesaikan dengan Linear Programming. Namun, kenapa tidak direkomendasikan untuk digunakan?


Masalah transportasi tidak direkomendasikan untuk dapat diselesaikan dengan Linear Programming karena pada penerapan dalam transportasi dan masalah karena transportasi produk yang bersifat homogen dari berbagai sumber atau supply ke berbagai tempat penjualan produk (demand). Alokasi produk dari daerah supply ke daerah demand ini dilakukan dengan tujuan untuk memaksimalkan laba dari penjualan produk atau alternatifnya adalah dengan meminimalkan biaya transportasi produk tersebut.

Pembahasan:

Metode Transportasi merupakan  suatu metode yang digunakan untuk mengatur distribusi dari sumber-sumber yang menyediakan produk-produk yang sama di tempat yang membutuhkannya secara optimal. Alokasi produk ini harus diatur dan dikelola dengan sedemikian rupa karena terdapat perbedaan biaya transportasi (alokasi) dari suatu sumber ke beberapa tujuan yang berbeda-beda dan dari beberapa sumber ke suatu tujuannya juga berbeda-beda.

Masalah transportasi tidak direkomendasikan untuk dapat diselesaikan dengan Linear Programming karena pada penerapan dalam transportasi dan masalah karena transportasi produk yang bersifat homogen dari berbagai sumber ke berbagai tempat penjualan produk. Alokasi produk dari daerah supply ke daerah demand ini dilakukan dengan tujuan untuk memaksimalkan laba dari penjualan produk atau alternatifnya adalah dengan meminimalkan biaya transportasi produk tersebut.

Metode ini lebih sederhana cara penggunaannya, karena tidak memerlukan closed path (salur tertutup). Dilakukan dengan cara mencari selisih dari biaya terkecil dengan biaya terkecil berikutnya untuk setiap kolom maupun baris selanjutnya. kemudian pilih selisih biaya terbesar dan alokasikan produk sebanyak mungkin ke sel yang memiliki biaya yang terkecil. Cara ini dilakukan secara berulang hingga semua produk sudah dialokasikan dengan benar.

3. Cara menghitung keuntungan maksimum menggunakan metode linear programming


Cara menghitung keuntungan maksimum menggunakan metode linear programming ada empat tahapan. Tahapan tersebut diantaranya menyederhanakan soal dengan mengubahnya kedalam bentuk model matematika, membuat fungsi dan pertidaksamaan, menggambar grafik dan menentukan titik potong, dan menghitung nilai fungsi berdasarkan titik potong.

Pembahasan

Metode linear programming adalah suatu metode untuk menyelesaikan masalah dengan menggunakan persamaan maupun pertidaksamaan linear dengan banyak penyelesaian, dengan memperhatikan syarat-syarat agar memperoleh hasil yang maksimum/minimum. Secara umum, masalah program linear dapat diselesaikan dengan langkah-langkah sebagai berikut.

Mengubah persoalan kedalam bentuk model matematika. Hal ini bertujuan untuk menyederhanakan soal agar lebih mudah dipahami.Membuat sistem pertidaksamaan dan fungsi tujuan berdasarkan model matematika.Menggambar grafik dan menentukan titik potongnya.Menghitung nilai fungsi tujuan berdasarkan titik potong yang ditentukan.Berdasarkan nilai fungsi tujuan ini, maka program linear telah diselesaikan.Pelajari lebih lanjut

36. kalau ada yang bisa programming, yg sudah pernah belajar for i in range(): tolong bantu, ada pr: Buatlah program yang akan menghitung jumlah bilangan dari 0 sampai 100 (termasuk angka100) dengan menggunakan perulangan for.


Kode program Python sederhana yang menghitung jumlah bilangan dari 0 sampai 100 (termasuk angka 100) dengan menggunakan perulangan for adalah:

[tex]\boxed{\,\begin{aligned}&\tt x = 0\\&\tt for\ i\ in\ range(1, 101):\\&\qquad\tt x = x+i\\ &\tt print("x =",\ x)\end{aligned}\,}[/tex]

dengan output eksekusi:
x = 5050
 

Pembahasan

Perintah [tex]\tt for\ i\ in\ range(a, b)[/tex] adalah salah satu jenis perulangan dalam bahasa pemrograman Python, yang akan melakukan aksi di dalam blok [tex]\tt for[/tex] ini dalam range [tex]\tt i=a[/tex] hingga [tex]\tt i=b-1[/tex] (bukan [tex]\tt i=b[/tex]).

Oleh karena itu, untuk persoalan ini, batas atas dalam range adalah 101, bukan 100. Untuk batas bawah, jika kita sudah melakukan inisialisasi variabel dengan 0, maka kita dapat menggunakan batas bawah 1, agar program tidak melakukan operasi penjumlahan 0+0 pada awal eksekusi perulangan [tex]\tt for[/tex].

Contoh kode programnya telah ditunjukkan di atas.

Program tersebut akan mengulangi penjumlahan [tex]\tt x+i[/tex] dan menyimpan nilainya dalam variabel [tex]\tt x[/tex], untuk digunakan kembali pada operasi [tex]\tt x+i[/tex] dengan nilai [tex]\tt i[/tex] yang sudah bertambah 1, hingga [tex]\tt i[/tex] bernilai 100.

Jika kita periksa dengan rumus deret aritmetika:

[tex]\begin{aligned}x&=0+S_{100}\\&=\frac{100(1+100)}{2}\\&=100(50{,}5)\\x&=5050\ \Rightarrow \sf sesuai\end{aligned}[/tex]

40. Apa manfaat dari mempelajari linear programming dan manfaat bagi kegiatan bisnis?


Manfaat dari mempelajari program linier yaitu untuk menyelesaikan masalah optimal. Sedangkan dalam bisnis manfaat dari memperlajari program linier yaitu menentukan nilai minimum dalam hal menentukan biaya sedangkan nilai maksimum dalam hal menentukan keuntungan.

Pembahasan :

Linier merupakan suatu data yang berada dalam satu garis bidang lurus yang sesuai dengan bidangnya masing-masing.

Program linier adalah salah satu program matematika dalam hal membagi sumber daya yang terbatas untuk mencapao tujuan. Program Linier digunakan untuk menyelesaikan masalah. Didalam lingkungan industri masalah tersebut dapat meliputi : perbankan, Kehutanan, Permunyakan, kehutanan, pendidikan dan lain sebagainya.

Manfaat dari mempelajari program linier yaitu untuk menyelesaikan masalah optimal. Sedangkan dalam bisnis manfaat dari memperlajari program linier yaitu menentukan nilai minimum dalam hal menentukan biaya sedangkan nilai maksimum dalam hal menentukan keuntungan. Selain itu didalam Metode Pemprograman Linier terdapat 2 jenis metode antara lain : Metode Grafik dan juga Metode Simpleks.
