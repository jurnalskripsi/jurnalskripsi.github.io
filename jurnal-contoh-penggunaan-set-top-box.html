1. Hasil pemeriksaan terhadap kualitas 15 Set Top Box di sebuah toko elektronik ditemukan 3 Set Top Box berkualitas buruk. Tentukan frekuensi relatif dari Set Top Box yang berkualitas baik!​


Jawaban:

Jumlah Set Top Box berkualitas baik = Total Set Top Box - Set Top Box berkualitas buruk

= 15 - 3

= 12

Frekuensi relatif adalah perbandingan antara jumlah kejadian tertentu dengan jumlah total kejadian. Dalam konteks ini, frekuensi relatif dari Set Top Box yang berkualitas baik dapat dihitung dengan membagi jumlah Set Top Box yang berkualitas baik dengan jumlah total Set Top Box yang diperiksa:

Frekuensi relatif Set Top Box berkualitas baik = (Jumlah Set Top Box berkualitas baik) / (Total Set Top Box)

= 12 / 15

= 0.8 atau 80%

Jadi, frekuensi relatif dari Set Top Box yang berkualitas baik adalah 80%.

2. hasil pemeriksaan terhadap kualitas 15 set top box Di sebuah toko elektronik Di temukan 3 set top box berkualitas buruk tentukan frekuensi relatif Dari set top box Yg berkualitas baik? ​


Jawaban:

Penjelasan dengan langkah-langkah:

Dalam kasus ini, ada total 15 set top box yang diperiksa. Dari pemeriksaan tersebut, ditemukan 3 set top box berkualitas buruk. Untuk menentukan frekuensi relatif dari set top box yang berkualitas baik, kita perlu mengetahui berapa jumlah set top box yang berkualitas baik.

Jumlah set top box yang berkualitas baik = Total set top box - Set top box berkualitas buruk

Jumlah set top box yang berkualitas baik = 15 - 3 = 12

Frekuensi relatif adalah rasio antara frekuensi observasi tertentu dengan total frekuensi atau jumlah observasi.

Frekuensi relatif dari set top box yang berkualitas baik = Jumlah set top box berkualitas baik / Total set top box

Frekuensi relatif dari set top box yang berkualitas baik = 12 / 15 = 0.8

Jadi, frekuensi relatif dari set top box yang berkualitas baik adalah 0.8 atau 80%.

4. Apa Keuntungan dan Kerugian dalalm Penggunaan List Box dan Combo Box !


Keuntungan dan kerugian dalam memakai  List Box dan Combo Box adalah untuk keuntungan maka kita dapat memberikan opsi untuk memberikan kontrol objek lain. Untuk kekurangan maka diperlukan sebuah algoritma yang dapat menangkap kontrol dari list dan combo box tersebut.

Pembahasan

Berbeda dengan objek kontrol lainnya, kotak kombo lebih sering digunakan sebagai variabel kontrol. Dengan menggunakan variabel kontrol, ini tidak terkait dengan tipe data tertentu (secara default, tipe data kotak kombo adalah CString). Selain itu,  semua fungsi yang terkait dengan kotak kombo, seperti fungsi GetCurSel() dan GetLBText(), dapat diakses menggunakan variabel kontrol. Kotak daftar memiliki fungsi yang hampir sama dengan kotak kombo yang menampilkan daftar. Perbedaannya adalah kotak kombo hanya menampilkan satu opsi sebelum Anda mengkliknya, sedangkan kotak daftar dapat menampilkan beberapa opsi. Kotak kombo adalah kontrol Microsoft Access yang bisa Anda gunakan dalam proses entri data yang memungkinkan Anda memilih data dari  daftar.

15. petunjuk penggunaan lap top


Laptop atau notebook yang semakin bertambah mumpuni, dan mudah dibawa menjadi alternatif untuk ponsel cerdas dan komputer meja demi menyelesaikan pekerjaan penting dimana saja kamu berada. Bila kamu baru saja menggunakan laptop atau baru saja menemuinya dan kamu tidak terbiasa, mungkin akan terlihat sedikit aneh pada awalnya. Jangan takut – ikuti langkah berikut ini untuk dapat mengoperasikan semua jenis laptop dan kami akan membuat anda dapat mengoperasikannya seperti seorang profesional dalam waktu singkat.

Bagian 1 dari 4: 
Mengatur Laptop AndaSunting

1

Bila kamu menggunakan laptop dirumahmu, carilah sebuah stop kontak dan colok charger(pengisi daya baterai) laptopmu. Laptop berfungsi menggunakan baterai yang dapat terkuras dengan cepat, khususnya bila kamu menggunakan laptop secara terus-menerus. Kecuali bila kamu berada di suatu tempat terkecil atau daerah asing dimana kamu harus pergi tanpa charger laptop, lebih baik untuk tetap mencolok charger dan mengisi baterai pada laptopmu.

2

Taruh bagian bawah dari laptopmu di meja tempat kamu duduk saat ini dan pastikan kamu duduk mengarah ke arah laptop. Disebut "laptop" karena mereka dapat digunakan pada pangkuanmu, tetapi bukan berarti pangkuan selalu menjadi tempat terbaik atau tempat yang tepat. Cobalah untuk mencari sudut yang nyaman bagi pergelangan tangan dan tanganmu – yang berarti kamu menggerakan laptopmu sampai kamu menemukan posisi terbaik untukmu.

Jangan menaruh laptopmu pada permukaan yang lembut, tidak beraturan atau kasar yang dapat menutup lubang udara dari laptop. Kebanyakan laptop mempunyai lubang udara yang berlokasi pada sisi dan bawah laptop yang harus dibiarkan terbuka agar laptop tetap dapat berfungsi dengan baik.

3

Mengangkat layar laptop sampai layar terlihat nyaman untuk dilihat.Kebanyakan laptop mempunyai pengunci yang memungkinkan bagian layar laptop untuk dibuka.

Bila laptop tidak dapat dibuka, jangan dipaksa! Cobalah mencari penguncinya. Kamu tidak harus memaksanya untuk membuka layar laptopmu.

Jangan menarik tutup laptop terlalu jauh. Sudut 45 derajat adalah sudut dimana kebanyakan laptop dibuka. Tutup atau mekanisme engsel mungkin saja rusak atau patah bila ditarik lebih jauh.

4

Carilah tombol daya dan nyalakan laptop. Pada kebanyakan laptop, tombol daya berlokasi sedikit di belakang keyboard(papan tombol). Tombol daya biasanya ditandai dengan simbol universal untuk 'menyalakan daya', sebuah lingkaran dengan garis dan melewati setengah bagian lingkaran.

5

Tunggu laptop sampai proses boot up(pengebutan) selesai. Karena laptop dirancang untuk mudah dibawa kemanapun dan juga daya baterai pada laptop, laptopmu mungkin mempunyai perangkat keras khusus yang menyebabkan laptop membutuhkan waktu lebih lama dalam proses boot up dibandingkan dengan komputer meja atau ponsel pintar.

6

Gunakan pointing device(peranti petunjuk) pada laptop. Pada kebanyakan komputer, bentuknya datar, daerah yang sensitif terhadap sentuhan disebut trackpad yang memungkinkan anda menggunakan jari sebagai pengganti mouse(tetikus). Cukup geser satu jari pada daerah trackpad untuk menggerakan kursor.

Banyak trackpad mendukung multi sentuh – dengan menggunakan lebih dari satu jari akan menghasilkan perbedaan aksi pada layar pemakai dibandingkan penggunaan menggunakan satu jari. Lakukanlah percobaan dengan laptopmu dengan menyeret salah satu jari, dua atau tiga jari di trackpad dan cobalah 'gerakan' berbeda atau pergerakan dengan jarimu.

Laptop Lenovo mungkin menggunakan perangkat kecil, joystick berwarna merah yang mirip seperti tombol dan disebut dengan nama "trackpoint" berlokasi di tengah keyboard diantara tombol huruf 'G' dan 'H'. Menggunakan trackpoint seperti menggunakan joystick yang sangat sensitif dan hanya dapat digunakan oleh jarimu.

Beberapa laptop tua mungkin masih memakai trackball(Bola jejak). Menggulirkan bola pada trackball akan menyebabkan mouse pointer bergerak.

Beberapa laptop dilengkapi dengan pen interface (antarmuka pena). Sebuah pena terlampir bersamaan di laptop pada kasus ini. Dengan melayangkan pena diatas layar akan menggerakan pointer, dan tekan pena untuk melakukan klik pada layar laptop.

Apakah kamu menemui kesulitan pada perangkat penunjuk kecil untuk digunakan? Kamu selalu dapat menambahkan sebuah mouse untuk digunakan pada laptopmu. Carilah USB port dan masukkan USB mouse ke port USB bila kamu ingin menggunakan mouse pada laptopmu. Laptop secara otomatis akan mengenali mouse dan membuat mouse siap untuk digunakan

7
Gunakan tombol klik kiri pada trackpad sebagai tombol utama mouse. Pada kebanyakan trackpad, untuk melakukan klik kamu dapat menggunakan sebuah tombol yang bertempat di bagian bawah kiri dari trackpad.

Beberapa trackpad membuatmu dapat melakukan klik dengan menekan secara pelan pada permukaan trackpad. Percobaan – kamu mungkin menemukan fungsi tambahan pada laptopmu yang tidak pernah kamu ketahui sebelumnya.

19. Sebutkan kelebihan menggunakan Virtual box dan kekurangan ketika tidak menggunakan Virtual box


jawab :

kelebihan virtual box :

1. Mudah digunakan, menyediakan pengaturan-pengaturan lebih lanjut;
2. Gratis;
3. Lebih ringan dibandingkan dengan aplikasi sejenis seperti Vmware;
4. Mendukung lebih banyak virtual hardisk;
5. Video memory (vga) dan performa cpu bisa diatur persentasenya.

kekurangan vitrual box :

1.Tidak bisa mengakses perangkat yang membutuhkan koneksi USB dengan sempurna;

2. Beberapa game yang fullscreen tidak berjalan.;

3. Membutuhkan memory yang besar, dan anda harus pandai-pandai mengatur resource memory untuk setiap virtual machine ( jika beberapa virtual machine dijalankan sekaligus ). Low / kekurangan memory dapat membuat semua virtual machine yang sedang running menjadi hang / crash;

4. Fitur drag n drop tidak berfungsi, pada banyak virtual machine;

5. Network ID untuk satu buah virtual machine, hanya 4 buah.

28. pada suatu pameran kuliner,salsa menjual rice box dalam tiga jenis yaitu premium set,simple set dan kids set. premium dijual dengan rp. 33.000,00, simple 24.000,00, dan kids set 9.000,00. pada pameran tersebut, simple set dan kids set yang terjual 30 lebih banyak dari setengah premium set yang terjual. jumlah simple yang terjual 5 lebih banyak dari 4 kali jumlah kids set yang terjual.jika jumlah hasil penjualan rice box seluruhnya rp.89.820.000,00, maka simple set yg terjual sebanyak ?​


Jawab:

Mari kita sebut jumlah premium set yang terjual sebagai x.

Dari soal, kita tahu bahwa jumlah simple set dan kids set yang terjual 30 lebih banyak dari setengah premium set yang terjual. Dengan kata lain,

jumlah simple set dan kids set yang terjual = 1.5x + 30

Kita juga tahu bahwa jumlah simple set yang terjual 5 lebih banyak dari 4 kali jumlah kids set yang terjual. Dengan kata lain,

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5

Kita bisa menggunakan kedua persamaan di atas untuk menyelesaikan masalah ini. Kita tahu bahwa total penjualan dari semua jenis rice box adalah Rp. 89.820.000,00, sehingga kita dapat menuliskan persamaan:

33.000x + 24.000(1.5x + 30) + 9.000(4x + 1) = 89.820.000

Setelah menyelesaikan persamaan tersebut, kita akan mendapatkan x = 1.800.

Kita dapat menggunakan nilai x tersebut untuk mencari jumlah simple set yang terjual:

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5 = 4 * (1.800 - 30)/5 + 5 = 573

Jadi, jumlah simple set yang terjual adalah 573.
