       cout << "\nArray ke-" << i << ": ";

       cin >> arrayKe[i];

   }

   

   cout << "\nMasukkan key yang ingin dicari: ";

   cin >> keyDicari;

   

   while(j <= elemenArray) {

       j++;

       if(arrayKe[j] == keyDicari) {

           cout << "\nKey ada di posisi index ke: " << j << endl;

           break;

       } else if(j == elemenArray) {

           cout << "\nKey tidak ada di posisi index manapun." << endl;

       }

14. Salah satu contoh dari penggunaan array adalah?


Contoh Program C++ Array 1 Dimensi

Berikut cara mendeklarasikan array 1 dimensi pada C++:

tipedata [totalindex]; //atau tipedata [] = { nilai1, nilai2, dst };

Jadi, ada dua cara mendeklarasikan sebuah array berdimensi satu cara pertama cukup dengan menggunakan tipe data dan total index yang diinginkan, contoh:

int umur[5];

Artinya, anda menyediakan varibel array bernama umur dengan tipe data integer yang mampu menampung 5 data yang sama mulai dari index 0 - 4.

Ingat: Index pada array selalu dimulai dari nol.

Kemudian untuk mengisi nilai untuk tiap index pada array dengan cara:

umur[0] = 18; umur[1] = 19; umur[2] = 20; umur[3] = 21; umur[4] = 22;

Cara kedua, anda bisa langsung memberi nilai untuk tiap index pada saat pendeklarasian array. Contoh kodenya sebagai berikut:

int umur[] = {18,19,20,21,22};

Untuk penerapan array dalam program biasanya dipadukan dengan perulangan for, while-do, dan do-while. Pilih salah satu namun yang paling umum adalah for.

Berikut contoh program sederhana array 1 dimensi pada c++:

#include <iostream> using namespace std; int umur[] = {18, 19, 20, 21, 22}; int n, hasil=0; int main () { for ( n=0 ; n<5 ; ++n ) { hasil+= umur[n]; } cout << hasil; return 0; }

Dalam program diatas, saya ingin menjumlahkan nilai dari tiap index umur dan hasilnya saya simpan kedalam variabel hasil untuk ditampilkan di akhir program.

17. Apakah contoh dari Array dalam Informatika


Jawaban:

Dalam ilmu informatika, Array adalah struktur data yang digunakan untuk menyimpan kumpulan elemen data dengan tipe yang sama, yang diakses menggunakan indeks. Berikut adalah beberapa contoh penggunaan Array dalam informatika:

1. Array dalam Bahasa Pemrograman:

  Dalam bahasa pemrograman seperti C, C++, Java, Python, dan sebagainya, Anda dapat membuat Array untuk menyimpan data seperti angka, huruf, atau bahkan objek. Misalnya, Anda dapat membuat Array untuk menyimpan daftar nilai siswa, data suhu dalam sehari, atau karakter dalam sebuah string.

2. Array dalam Spreadsheet:

  Dalam program spreadsheet seperti Microsoft Excel atau Google Sheets, Anda dapat menggunakan Array untuk menyimpan data dalam bentuk tabel. Misalnya, Anda dapat menggunakan Array untuk menyimpan data penjualan per bulan atau data stok barang.

3. Array dalam Gambar Digital:

  Dalam pemrosesan gambar digital, Array sering digunakan untuk menyimpan piksel-piksel gambar. Setiap elemen dalam Array mewakili nilai piksel (misalnya, warna atau tingkat kecerahan) pada lokasi tertentu dalam gambar.

4. Array dalam Database:

  Dalam basis data, Array digunakan untuk menyimpan data terstruktur dalam tabel. Misalnya, dalam MySQL atau PostgreSQL, Anda dapat menggunakan Array untuk menyimpan daftar nilai dalam satu kolom dari suatu tabel.

5. Array dalam Pengolahan Sinyal:

  Dalam pengolahan sinyal atau audio, Array digunakan untuk menyimpan sampel-sampel suara dalam bentuk digital. Setiap elemen Array mewakili nilai amplitudo pada suatu waktu tertentu.

Contoh-contoh di atas menunjukkan bahwa Array adalah struktur data yang fleksibel dan dapat digunakan dalam berbagai bidang untuk mengatur dan menyimpan data dengan efisien.

Pendahuluan

Matriks adalah susunan bilangan atau simbol yang disusun dalam bentuk baris dan kolom sehingga membentuk suatu bangun persegi.

Jenis –jenis Matriks

• Matriks Baris

adalah matriks yang hanya terdiri satu baris.

• Matriks Kolom

adalah matriks yang hanya terdiri satu kolom.

• Matriks Nol

adalah matriks yang semua elemennya berupa bilangan nol.

• Matriks Persegi

Matriks ini biasanya terjadi ketika jumlah baris sama dengan jumlah kolom.

• Matriks Diagonal

adalah matriks persegi dengan bilangan pada posisi diagonalnya tidak nol.

24. jelaskan definisi dari array multidimensi dan berikan contoh array multidimensi yang dideklarasikan dan di gunakan dalam perhitungan?​


Array multidimensi adalah kumpulan data yang tersusun dalam bentuk matriks atau tabel dua dimensi, atau lebih dari dua dimensi. Setiap elemen dalam array multidimensi dapat diakses melalui indeks yang terdiri dari beberapa nilai, tergantung pada dimensi array tersebut.

Contoh array multidimensi yang dideklarasikan dan digunakan dalam perhitungan adalah array 2x3 yang menyimpan nilai-nilai dalam matriks:

int[][] matrix = {{1, 2, 3}, {4, 5, 6}};

Dalam contoh ini, kita mendeklarasikan array multidimensi dengan 2 baris dan 3 kolom. Setiap elemen dalam matriks ini dapat diakses menggunakan indeks dua dimensi. Misalnya, untuk mengakses nilai 4, kita dapat menggunakan matrix[1][0] karena 1 adalah indeks baris dan 0 adalah indeks kolom.
