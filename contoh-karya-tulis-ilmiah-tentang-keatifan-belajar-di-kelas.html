Jawaban:

Cara 1 ( Icon Desktop ) :

1. Klik 2x pada icon Microsoft Word yang ada di desktop

Cara 2 ( Recent List ) :

1. Klik Windows Button, atau dengan menekan tombol Windows yang ada di keyboard ( jika ada )

2. Pada saat Menu Windows dibuka akan menampilkan Recent List ( jika ada ), klik Microsoft Office Word

Cara 3 ( Search ) :

1. Klik Windows Button, atau dengan menekan tombol Windows yang ada di keyboard ( jika ada )

2. Ketikan "Office Word" atau "winword" pada kolom pencarian

3. Jika muncul aplikasi dengan icon MS word, klik aplikasi tersebut

Cara 4 ( Run Program ) :

1. Tekan Tombol Windows + R, jika tidak ada maka klik Windows Button lalu cari dan klik "Run.."

2. Setelah ada tampilan dialog "Run", masukan kata "winword"

3. Klik ok atau tekan ENTER

Cara 5 ( Program Data ) :

1. Klik Windows Button, atau dengan menekan tombol Windows yang ada di keyboard ( jika ada )

2. Klik All Program

3. Lalu klik folder Microsoft Office

4. Klik aplikasi Microsoft Office World

37. tuliskan 3 ciri ciri dari masing masing atif di bawah ini A. otot polosB. otot lurikC. otot jantung​


Jawaban:

otot polos

Berbentuk gelondong, kedua ujungnya meruncing dan bagian tengan menggelembung.

Memiliki satu inti sel.

Tidak memiliki garis melintang.

otot lurik

Sel otot lurik berbentuk silinder yang memiliki bagian gelap dan terang sehingga tampak seperti lurik.

Memiliki inti banyak.

Otot bekerja atas kesadaran perintas otakn dan dapat mengalami kelelahan.

otot jantung

Sel otot jantung berbentuk serabut lurik yang bercabang-cabang.

Sel mempunyai satu atau banyak inti sel dan terletak di tengah serabut.

Otot bekerja di luar kesadaran alias tidak diperintah otak, tapi dipengaruhi oleh persediaan oksigen yang cukup.

Jawaban:

A. Otot polos:

Bentuk: ujungnya runcing berbentuk gelondong

Inti: satu ditengah

letak: menyusun semua organ dalam kecuali jantung

sifat kerja: dipengaruhi oleh syaraf tak sadar, reaksi terhadap rangsang lambat, namun tak cepat lelah

B. Otot lurik:

bentuk: adanya garis gelap terang(serat lintang), bentuk silindris

inti: banyak di pinggir

letak: melekat pada rangka

sifat kerja: dipengaruhi oleh syaraf sadar, reaksi terhadap tanggapan cepat, namun cepat lelah

C. Otot jantung:

bentuk: adanya garis gelap terang (serat lintang), bentuk silindris dan serabut yang bercabang-cabang

inti: banyak di tengah

letak: pada dinding jantung

sifat kerja: dipengaruhi oleh syaraf tak sadar, bekerja secara otomatis dan ritmis.

40. contoh mengatifkan progam ekstrakulikuler yang ada di sekolah dan terlibat aktif untuk membangun kepengurusan yang harmonis.​


Jawaban:

Berikut adalah contoh cara mengaktifkan program ekstrakurikuler di sekolah dan terlibat aktif untuk membangun kepengurusan yang harmonis:

1. Menjalin komunikasi dengan siswa dan guru yang terlibat dalam program ekstrakurikuler. Kamu bisa membuat grup chat atau forum diskusi untuk memudahkan komunikasi dan koordinasi.

2. Mengadakan rapat pengurus secara rutin untuk membahas program dan kegiatan yang akan dilakukan. Dalam rapat ini, semua pengurus dapat memberikan masukan dan ide untuk memperbaiki program ekstrakurikuler.

3. Mengadakan kegiatan yang menarik dan bermanfaat untuk siswa, seperti workshop, seminar, atau pertandingan. Kegiatan ini dapat menarik minat siswa dan membantu mereka mengembangkan keterampilan baru.

4. Membuat jadwal kegiatan yang jelas dan teratur, sehingga siswa dapat mempersiapkan diri dengan baik dan menghindari bentrok dengan jadwal kegiatan lain.

5. Mempromosikan program ekstrakurikuler di sekolah dan media sosial, sehingga siswa dapat mengetahui program yang ada dan tertarik untuk bergabung.

6. Membuat evaluasi program secara berkala untuk mengetahui keberhasilan program dan mengevaluasi kekurangan yang ada. Dari evaluasi ini, kamu dapat memperbaiki program dan meningkatkan kualitas kepengurusan.

Dengan mengaktifkan program ekstrakurikuler dan terlibat aktif dalam membangun kepengurusan yang harmonis, kamu dapat membantu meningkatkan kualitas pendidikan di sekolah dan memberikan pengalaman berharga bagi siswa. Selain itu, kamu juga dapat mengembangkan keterampilan kepemimpinan dan kerjasama yang berguna untuk masa depan.
