19. Jelaskan perbedaan sistem operasi dengan program aplikasi dalam sebuah sistem komputer ? Jelaskan dan berikan contoh-contohnya !.


Perbedaan sistem operasi dan program aplikasi

Segi Tugas

Tugas sistem operasi sendiri adalah mengatur manajemen resources dan operasi yang terjadi antar komponennya.

Program aplikasi bertugas untuk memenuhi kebutuhan penggunanya melalui kegunaannya baik secara umum maupun spesifik

Segi Keterkaitan

Sistem operasi dapat berjalan walaupun tidak ada program aplikasi. Akan tetapi karena tidak ada program aplikasi, sistem operasi tidak memiliki manfaat untuk penggunanya

Program aplikasi hanya dapat berjalan jika ada sistem operasi yang aktif di komputer tersebut

Contoh

Sistem operasi yang populer saat ini,Microsoft Windows, Macintosh dan Linux

Program aplikasi seperti program aplikasi pengolah grafis ( Adobe photoshop, CorelDrwa, AutoCad), pengolah kata ( Microsoft Word, WPS ) dan masih banyak lagi.

Pembahasan

Sistem operasi adalah sekumpulan program yang memiliki tugas dalam penagturan resources sekaligus menjadi jembatan perangkat keras dengan perangkat lunak. Keberadaan sistem operasi dalam suatu sistem komputer sangat diperlukan.

Program aplikasi atau Sistem Aplikasi merupakan program yang sudah berbentuk jadi yang memiliki fitur dan tujuan yang berbeda - beda sesuai kebutuhan pengguna dan kemampuan komputer. Program aplikasi secara umum contohnya pengolah kata, pengolah angka, pengolah database, pengolah grafis dan lain - lain. Sedangkan secara spesifik seperti Microsoft Word, Microsoft Access, Adobe After Effect, Open Office Calc dan masih banyak lagi

38. jelaskan apa yang Anda ketahui tentang sistem operasi dan program aplikasi, berikan contohnya.


Pembahasan
Sistem operasi adalah sekumpulan program yang memiliki tugas dalam penagturan resources sekaligus menjadi jembatan perangkat keras dengan perangkat lunak. Keberadaan sistem operasi dalam suatu sistem komputer sangat diperlukan.

Program aplikasi atau Sistem Aplikasi merupakan program yang sudah berbentuk jadi yang memiliki fitur dan tujuan yang berbeda - beda sesuai kebutuhan pengguna dan kemampuan komputer.

Program aplikasi secara umum contohnya pengolah kata, pengolah angka, pengolah database, pengolah grafis dan lain - lain. Sedangkan secara spesifik seperti Microsoft Word, Microsoft Access, Adobe After Effect, Open Office Calc dan masih banyak lagi
