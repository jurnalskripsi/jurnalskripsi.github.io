17. dalam HTML dapat menambahkan hubungan atau relasi ke halaman web tertentu. berikut ini syntax HTML hyperlink yang apabila di klik akan menuju ke halaman web www.google.com adalah


Pembahasan :

Dalam HTML dapat menambahkan hubungan atau relasi ke halaman web tertentu. Syntax HTML hyperlink yang apabila di klik akan menuju ke halaman www.google.com adlah dengan menggunakan instruksi tag <a>.

ketika di klik maka akan berpindah ke halaman lain seprti pada contoh www.google.com ini merupakan tujuan dari hypertext pada HTML. Pada tag <a> terdapat tulisan <a> yang memiliki makna adalah sebagai ancor yaitu untuk mencari, pada setiap tag a mempunyai atribut berupa href. Hypertext experience adalah singkatan dari href pada atribut. Pada contoh di bawah ini :

<!Doctype html>

<html>

<head>

         <title> syntax HTML </title>

</head>

<body>

          <hi> menuju ke halaman web </hi>

           <p> Belajar </p>

           <a href ="http://www.google.com"> Pembelajaran </a></p>

</body>

</html>

1. Jelaskan cara membuat daftar/list di html?

cara untuk membuat daftar/list di html kamu bisa mengunakan <ol>, <ul> dan <dl> untuk menentukan tipe listnya dan list mengunakan <li>(untuk <ol> dan <ul>) atau <dt> dan <dd>(untuk <dl>) untuk menyatakan bahwa teks didalamnya adalah bagian dari daftar atau listnya.

<ol> adalah list yang mempunyai urutannya contohnya

1. daftar 1

2. daftar 2

3. daftar 3

sedangkan <ol> adalah list yang tidak mempunyai urutannya contohnya

- daftar 1

- daftar 2

- daftar 3

sedangkan <dl> adalah list yang mempunyai deskripsinya, <dt> adalah sesuatu yang akan di deskripsikan sedangkan <dd> adalah deskripsinya contohnya

Kopi

    -minuman panas berwarna hitam

Susu

    -minuman dingin berwarna putih
