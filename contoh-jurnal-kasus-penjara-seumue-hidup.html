3. Jamil arshad di penjara di malaysia dalam kasus apa ?


Tidak disebutkan pasti kasus apa yang menjerat Jamil Arshad sehingga menjadi narapidana Indonesia yang paling lama dipenjara di Malaysia. Namun dari pasal yang menjerat (Pasal 5 Undang-Undang Se.nja.ta Api), kita bisa menyimpulkan bahwa ia di masa silam melakukan kejahatan penyalahgunaan sen.jata api yang berbuah hukuman seumur hidup.

Pembahasan

Pasal 5 Undang-Undang Se.njata Api (Peningkatan Hukuman) 1971 (UU 37) adalah sebuah pasal dalam undang-undang Malaysia yang mengatur tentang peningkatan hukuman bagi pelanggar undang-undang sen.jata api. Pasal ini memperberat hukuman untuk pelaku kejahatan yang menggunakan sen.jata api, termasuk penjara seumur hidup dan hukuman cam.buk.

Di Indonesia sendiri, penggunaan sen.jata api diatur dalam Undang-Undang Nomor 12 Tahun 1951 tentang Pendaftaran Sen.jata Api dan Bahan Peledak. Undang-undang ini kemudian diperbaharui dengan Undang-Undang Nomor 8 Tahun 2012 tentang Sen.jata Api dan Bahan Peledak. Menurut undang-undang tersebut, senj.ata api di Indonesia dibagi menjadi dua kategori, yaitu sen.jata api laras panjang dan senj.ata api laras pendek.

Jamil Arshad adalah nama yang akhir-akhir ini populer sebab kisanya dimunculkan ke publik oleh salah satu pesohor Indonesia. Jamil berasal dari NTB Indonesia dan menjadi pekerja migran di Malaysia. Sayangnya, ia di masa silam sering keluar masuk penjara dan terakhir didakwa hukuman seumur hidup. Namun baru menjalani masa hukuman 40 tahun, ia mendapat pengampunan dari Raja Malaysia dan dipulangkan ke Indonesia pada tahun 2023.

34. Kehidupan kebudayaan kejaraan majapahit ?


Kehidupan sosial dan budaya Majapahit pada masa Tribhuwanatunggadewi ini berjalan begitu bagus. Keadaan sosial berjalan sangat berimbang, setelah pada masa Jayawardhana masyarakat seperti tak diperhatikan.
