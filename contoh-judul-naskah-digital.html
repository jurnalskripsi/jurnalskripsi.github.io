Fungsi Naskah digital yaitu sebagai berikut:
a. Sebagai salah satu alternatif media belajar.
b. Sebagai media berbagi informasi.
c. Seseorang dengan sangat mudah dapat menjadi pengarang serta penerbit dari naskah yang dibuatnya sendiri.

Tujuan Naskah Digital yaitu sebagai berikut:
a. Memberikan kesempatan bagi yang membuat konten untuk lebih mudah berbagi informasi, dengan cara yang lebih menarik dan interaktif.

30. langkah langkah pembuatan naskah digital


Siapkan file word dengan ekstensi .doc atau .docx yang akan anda buat menjadi Buku Digital.Pastikan isi file word anda sudah rapi.

Jika file word anda mengandung gambar, pastikan semua gambar dengan format Text Wrapping In Line With Text. Cara mengaturnya, klik kanan pada gambar dan pilih Text Wrapping kemudian pilih In Line With Text

Kemudian, jika file anda mengandung tabel, pastikan semua tabel anda dengan Preffered width 100%. Cara mengaturnya, klik kanan pada tabel yang ingin anda atur, pilih Table Properties. Kemudian atur Preffered width

Jika sudah selesai, selanjutnya save as file anda dengan tipe file Web page Filtered

Kemudian, buka software Sigil.

Kemudian klik File>Open, kemudian pilih file html (web page filtered) yang sudah di save tadi.

Setelah file html tadi sudah tampil pada Sigil, cek kembali posisi dan kerapihan isi file.

Jika kita ingin menambahkan gambar pada file kita, pada kotak sebelah kiri ada folder Images. Klik kanan pilih Add Existing File. Kemudian pilih gambar yang ingin kita tambahkan.

Lalu pilih tempat pada file, dimana ingin menempatkan gambar yang sudah di upload tadi.

Klik kanan, pilih insert file.

Jika kita ingin menambahkan video pada Buku Digital kita, lakukan hal yang sama seperti memasukkan gambar pada file kita. Namun, format video yang dapat ditampilkan pada Buku Digital ini hanyalah berformat .mp4. Jika video tidak dapat dijalankan ketika sudah menjadi Buku Digital, ubah bitratenya menjadi lebih rendah dari sebelumnya.

Selanjutnya, kita tambahkan cover untuk Buku Digital kita. Caranya, sama seperti mengupload gambar pada folder Images. Setelah gambar sudah masuk pada folder Images, pada menu bar pilih Tools>Add Cover

Setelah itu pilih gambar yang akan kita gunakan sebagai Cover Buku Digital kita.

35. pemilihan format dan pemanfaatan naskah digital


1. Format yang paling portable dan bisa diakses oleh macam2 tipe komputer (PC, Android, Mac, Linux) adalah PDF
2. Format yang lebih modern dan khusus dirancang utk digital book adalah EPUB
3. Format digital e-book milik Amazon adalah AZW dan AZW3 yg khusus dirancang utk device Kindle dari Amazon juga
4. Format document office yang paling umum milik Microsoft adalah DOC dan DOCX
5. Format dokumen digital untuk jurnal bergengsi biasanya TeX atau LaTeX
6. Format MOBI dulu ada, tapi sejak 2006 telah dibeli oleh Amazon dan sudah tidak dikembangkan lebih lanjut sejak 2011
7. Format LIT milik Microsoft
8. Format DJVU, sangat efisien khusus untuk naskah digital hasil scan (biasanya dipakai utk buku2 kuno)
9. Format buku komik, biasanya CBZ, CB7, CBR, CBT dan CBA

39. tujuan di buat naskah digital


Ada beberapa tujuan pengembangan buku digital:
1) Memberikan kesempatan bagi pembuat konten untuk berbagi informasi dengan lebih mudah, dengan cara yang lebih menarik dan interaktif. Dengan membuat konten dalam bentuk digital, pengarang tidak perlu mendatangi penerbit untuk menerbitkan bukunya, cukup mendatangi salah satu situs penerbit digital dan memberikan bukunya secara mandiri.
2) Melindungi informasi yang disampaikan.
Berbeda dengan buku fisik yang dapat rusak, basah maupun hilang, buku digital berupa data di komputer terlindungi dari masalah-masalah tersebut. Walaupun data tersebut hilang, pengguna dapat dengan mudah mencari penggantinya baik dari internet maupun meminta kembali pada pembuat buku.
